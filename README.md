# Helix Jump 3D Game

A 3D arcade-style game inspired by Helix Jump, developed using **Unity 3D** and **C#**. The player controls a bouncing ball and navigates through a rotating helix tower while avoiding obstacles and reaching the bottom level.

## Features

* Smooth ball movement using Unity Physics Engine
* Rotating helix platform controls
* Collision detection and obstacle handling
* Multiple platform levels
* Score tracking system
* Game Over and Restart functionality
* Interactive 3D gameplay environment

## Technologies Used

* Unity 3D
* C#
* Unity Physics Engine
* Visual Studio

## Gameplay

The objective is to guide the ball safely through gaps in the rotating platforms while avoiding dangerous sections. The game becomes progressively challenging as the player advances through levels.

## Project Structure

* Scripts/ – Game logic and player controls
* Prefabs/ – Reusable game objects
* Scenes/ – Game scenes and levels
* Assets/ – Models, materials, textures, and audio files
